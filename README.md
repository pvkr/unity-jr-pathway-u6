# Unity Junior Programmer — Scene Flow & Data Sample Project

A sample project from the [Unity Learn Junior Programmer pathway](https://learn.unity.com/pathway/junior-programmer/unit/manage-scene-flow-and-data/tutorial/explore-the-sample-project?version=6.0), Unit: **Manage Scene Flow and Data**.

## Overview

This project demonstrates scene management and data persistence in Unity. It includes a simple resource-gathering simulation where units transport resources to a base, showcasing:

- Scene loading and transitions (Menu → Main)
- Data management between scenes
- Basic unit and building mechanics

## Project Structure

- `Assets/Scripts/` — Game logic: `Unit`, `Base`, `Building`, `ResourcePile`, `TransporterUnit`, `UserControl`, `MenuUIHandler`
- `Assets/Scenes/` — `Menu.unity` and `Main.unity` scenes
- `Assets/Prefabs/`, `Assets/Models/`, `Assets/Sprites/` — Art and prefab assets

## Requirements

- Unity 6000.0.68f1 (Unity 6)
